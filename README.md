# README

## Introdução
> Sistema instalado no wsl utilizando o RVM, ruby versão 3.0 e rails 7.0

> Utilização do Banco de Dados MySQL

Criação da aplicação com o seguinte comando:
```
rails _7.0_ new . --css=bootstrap -d=mysql -T
```
